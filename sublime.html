Sublime Text

Sublime Text is a text editor that code and text editor that has a lot of useful tools and features. The current version is Sublime Text 3.

I do most of my text-work with Sublime, to the point of it being a running joke that I will proselytize the program whenever I can.

  • You can make multiple cursors and edit multiple lines simultaneously, and select lines independant of each other
  • You can create a new line before or after the current line you are editing, without your cursor needing to be at the start/end of the line
  • You can indent or comment selected lines with a keyboard shortcut
  • You can select repeated instances of a selected word or phrase
  • You can instantly delete a line without having to select it first
  • You can shift lines up and down without cutting and pasting, or selecting
  • You can join a line with the one below it, making a single giant line separated by a space
  • You can wrap selected text with a tag, and edit both the opening and closing ends at the same time
  • You can have lay out multiple panes in a window, or multiple windows
  • And of course the classics like copy, undo, etc.
